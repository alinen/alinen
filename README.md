### Hi there 👋

I am an Associate Professor at Bryn Mawr College in the Computer Science department. Before coming to Bryn Mawr College, I 
worked primarily as a C++ programmer to build networking, simulation and visualization APIs and applications. 

I like to write games, interactive experiences, and low-level device demos and systems. I research avatars and procedural characters with a focus on body language and I'm also interested in neo-luddism, permacomputing, and the political economy of software in general.   

Website: [https://alinen.net](https://www.alinen.net)

## Repositories

**APIs, Tools, and Plugins**

* [open-body-fit](https://github.com/alinen/open-body-fit): Software for estimating biomechanics from poses extracted from video.
* [AGL](https://github.com/alinen/agl): C++ 3D/2D drawing API with built-in camera, primitives, shaders, texture, and text support (in openGL).
* [ATK](https://github.com/alinen/atk): C++ API and tools for loading skeletons and animations from BVH files.
* [tinygl](https://github.com/alinen/tinygl-cpp): Single header, cross-platform openGL framework for drawing 2D squares, circles, and triangles. 
* [bvh-python](https://github.com/alinen/bvh-python): Utilities for reading, writing, visualizing, and computing forward kinematics with BVH files with Python.
* [Curation Tree](https://github.com/alinen/curation-tree): Framework for behavior trees tailored for narrative and puzzle games.
* [MotionScriptTools](https://github.com/alinen/MotionScriptTools): Scripting tools for matlab, motion builder, and maya

**Games, demos, plugins, and applications**

* [Rubik's cube solver](https://github.com/alinen/cube): Heuristic-based approach for solving a Rubik's Cube. Visualized in Unity. 
* [QWOP](https://github.com/alinen/qwop): Remake of QWOP written in pymunk and pyglet
* [Song of Philadelphia Map](https://github.com/alinen/song-of-philadelphia-map): Map of locations referenced in the podcast "Song of Philadelphia" by Julien Suaudeau
* [Virtuality Theater](https://github.com/alinen/virtuality-puppets): A virtuality puppet maps a real world puppet to a virtual counterpart.
* [Attack of the Killer Vegetables](https://github.com/alinen/veggiegame): Simple arcade game written in HTML/Javascript.
* [Cloud Therapist](https://github.com/alinen/cloud-therapist): Eliza-inspired chatbot that runs in a browser.
* [Tangram game](https://github.com/tangram-game): Tangram game based on our plugin for [jsPsych](https://github.com/jspsych/jspsych-contrib)
* [Peg Board Experiment Platform](https://github.com/alinen/peg-board-exp-platform): Experiment that encorporates a peg board with a simple game.
* [Improv Chatbot](https://github.com/alinen/chatbot-improv): Chatbot based on the SPOLIN model
* [Fluid Simulation](https://github.com/alinen/fluids): C++ fluid simulation of smoke and fire.
* [Misc](https://github.com/alinen/misc): Graphics experiments in processing.
* [Shared Dreaming](https://github.com/alinen/shared-dreaming): Particle demos based on the LEAP Motion and webGL
* [Glitch](https://github.com/alinen/glitch): Wampus-inspired javascript game.
* [Manatee](https://github.com/alinen/manatee): Old Twitter bot that generates inspriational manatee messages 
* [Selection Word Count](https://github.com/alinen/selection-word-count): Plugin for VS Code

**Device software**

* [Digital Notebook](https://github.com/alinen/digital-notebook): A simple text editor for the esp32 that supports a small screen and SD memory.
* [Pi Pico Slideshow](https://github.com/alinen/pi-pico-slideshow): Simple slideshow application based on the Pi Pico
